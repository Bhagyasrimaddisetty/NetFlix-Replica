# Netflix Replica

A single-page clone of the Netflix landing/marketing page, built with plain HTML, CSS, and JavaScript — no frameworks or external libraries.

## Features

- Hero section with background imagery and call-to-action
- Email signup form to start a free trial
- Feature highlights section
- FAQ section
- Footer with language selector and support links

## Tech Stack

- HTML5
- CSS3 (inline, self-contained)
- Vanilla JavaScript

## Getting Started

Clone the repo and open `index.html` directly in a browser — no build step or dependencies required.

```bash
git clone https://github.com/Bhagyasrimaddisetty/NetFlix-Replica.git
cd NetFlix-Replica
```

Then open `index.html` in your browser.
